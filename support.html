Frequently Asked Questions

What is Supanator? ▼
Supanator is an independent client application for managing Supabase projects. It provides a mobile-friendly interface to interact with your Supabase databases, storage, Edge functions, and more.
How do I connect to my Supabase project? ▼
You can connect using your experimental API token, which allows automatic project discovery. Alternatively, you can use your project URL and service role key for direct connection.
Is my data secure? ▼
Yes, all connections are made directly to your Supabase project using secure HTTPS. We don't save any token data. Please refer to our privacy policy.
Where do I get my experimental API token? ▼
You can get your experimental API token from your Supabase Dashboard under Settings > API. Look for the 'Experimental' section and copy the token that starts with 'sbp_v0_'.
Why can't I see my projects? ▼
Make sure your experimental API token is correct and has the necessary permissions. Some tokens may have limited access to project discovery features.
Why can't I see my Edge functions? ▼
Edge functions need to be deployed to your Supabase project first. Use the Supabase CLI or Dashboard to deploy functions. They will then appear in the Edge functions tab.
What features require a Pro subscription? ▼
Pro features include SQL editor, storage management, edge functions, and analytics. The basic tables view is always free.
Is Supanator affiliated with Supabase? ▼
No, Supanator is an independent application created by a third-party developer. It is not affiliated with or endorsed by Supabase Inc.
